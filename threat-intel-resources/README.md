# 📚 Threat Intelligence Resources

A curated reference list maintained by [ThreatSignal](https://threatsignal.in). These are the frameworks, databases, and feeds we use in our research.

## Frameworks & Models

| Resource | Description | Link |
|----------|-------------|------|
| MITRE ATT&CK | Adversary tactics and techniques matrix | [attack.mitre.org](https://attack.mitre.org) |
| Diamond Model | Structured threat analysis model | [activeresponse.org](https://www.activeresponse.org/wp-content/uploads/2013/07/diamond.pdf) |
| Cyber Kill Chain | Lockheed Martin's intrusion model | [lockheedmartin.com](https://www.lockheedmartin.com/en-us/capabilities/cyber/cyber-kill-chain.html) |
| STIX/TAXII | Structured threat intel exchange standard | [oasis-open.org](https://oasis-open.github.io/cti-documentation/) |
| VERIS | Vocabulary for Event Recording and Incident Sharing | [veriscommunity.net](http://veriscommunity.net) |
