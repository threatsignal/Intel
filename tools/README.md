# 🛠️ Tools

This directory contains standalone tools and utility pipelines used to enrich, monitor, and analyze threat intelligence data.

## Available Tools

### 1. [Telegram Threat Monitor](telegram_threat_monitor/README.md)
An automated threat intelligence pipeline that monitors underground Telegram channels, extracts Indicators of Compromise (IOCs), translates foreign language messages using Google Gemini, and pushes alerts to Discord.

For setup and execution details, refer to the [Telegram Threat Monitor README](telegram_threat_monitor/README.md).
